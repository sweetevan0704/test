# 图片压缩工具

这是一个简单易用的在线图片压缩工具，具有苹果风格的界面设计。

## 功能特点

- 支持上传 PNG、JPG 等格式的图片
- 可调节压缩比例
- 实时预览压缩效果
- 显示压缩前后的文件大小
- 支持压缩后图片下载
- 响应式设计，支持各种设备

## 技术栈

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (原生)
- 图片压缩库：browser-image-compression

## 项目结构

```
/
├── index.html          # 主页面
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── main.js        # 主要逻辑
└── README.md          # 项目说明
```

## 使用说明

1. 点击上传区域或拖拽图片到指定区域
2. 选择需要的压缩比例
3. 等待压缩完成
4. 预览压缩效果
5. 下载压缩后的图片

## 浏览器支持

- Firefox
